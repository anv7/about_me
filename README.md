### 准备环境 

    1.安装 node.js  
    2.安装 gulp 全局的命令行接口  
    执行命令: `npm install gulp -g`

### 项目初始化 

    安装 node package modules
    执行命令: `npm install`    
    
### 开启服务器开始构建开发

    1.npm install
    
    2.gulp dd
    
### 构建产品级上线代码

    gulp dd -p
